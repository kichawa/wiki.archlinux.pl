Wicd

Z wiki.archlinux.pl

[edytuj] Wprowadzenie

Wicd jest menedżerem połączeń sieciowych, zarządza interfejsami sieci przewodowych i bezprzewodowych. Jest napisany w Pythonie i GTK+, co wymaga mniej zależności od innych zarządców sieci.

[edytuj] Instalacja

Instalacja Wicd: 

# pacman -S wicd

Jeśli potrzebujesz tylko GUI oraz ikonki w tacce systemowej: 

# pacman -S wicd-gtk

Jeżeli nie używasz środowiska GNOME, możesz także potrzebować: 

# pacman -S xfce4-notifyd

Zalecany jest dbus: 

# pacman -S dbus

[edytuj] Konfiguracja

Należy wyedytować plik /etc/rc.conf i w sekcji NETWORKING wyłączyć uruchamianie wszystkich interfejsów sieciowych:

File: /etc/rc.conf
# -----------------------------------------------------------------------
# NETWORKING
# -----------------------------------------------------------------------
#
# HOSTNAME: Hostname of machine. Should also be put in /etc/hosts
#
HOSTNAME="nazwa_hosta"

...

# interface=eth0
# address=
# netmask=
# gateway=

...

W sekcji DEAMONS wyłączamy wpis network i dopisujemy wicd i dbus:

File: /etc/rc.conf
DAEMONS=(...!network dbus wicd...)

Na koniec trzeba dodać użytkownika do wymaganej grupy: 

# gpasswd -a nazwa_użytkownika network

W przypadku połączenia dynamicznego potrzebny jest także dhcdbd, którego po zainstalowaniu także należy dodać do sekcji DAEMONS w pliku /etc/rc.conf.
