[edytuj] Wprowadzenie

Z Wikipedii:

SYSLINUX Project jest zestawem lekkich programów rozruchowych, służących do uruchamiania jądra Linux. Autorem programów jest H. Peter Anvin. W skład projektu wchodzą:

  • oryginalny SYSLINUX, używany do załadowania i uruchomienia Linux z systemu plików FAT (najczęściej stosowany do uruchamiania z dyskietek);
  • ISOLINUX, używany do załadowania i uruchomienia jądra Linux z systemu plików ISO 9660 na dyskach CD-ROM;
  • PXELINUX, używany do załadowania i uruchomienia jądra Linux z serwera sieciowego używając Pre-boot eXecution Environment (PXE);
  • EXTLINUX, używany do załadowania i uruchomienia jądra Linux z systemów plików ext2, ext3, ext4 oraz btrfs;
  • MEMDISK, używany do uruchomienia starszych systemów takich jak MS-DOS;
  • dwa oddzielne systemy menu; oraz
  • środowisko programistyczne dla dodatkowych modułów

SYSLINUX nie jest już używany do rozruchu systemów opartych o jądro Linux, częściej jest używany w różnego rodzaju dyskietkach ratunkowych oraz tzw. lekkich systemach. ISOLINUX jest często używany w dystrybucjach typu LiveCD.

Może on z powodzeniem zastąpić domyślne bootloadery GRUB czy GRUB2.

[edytuj] Instalacja

Notka: Skrypt syslinux-install_update zainstaluje Syslinux, skopiuje moduły COM32 do /boot/syslinux, ustawi flagę boot i zaktualizje MBR.

1. Syslinux zainstalujesz podając polecenie:

# pacman -S syslinux

2. Upewnij się, że partycja /boot jest zamontowana.

3. Uruchom skrypt syslinux-install_update z parametrami -i (zainstaluj syslinux) -a (ustaw flagę boot) -m (zainstaluj mbr):

# /usr/sbin/syslinux-install_update -iam

4. Wyedytuj plik /boot/syslinux/syslinux.cfg i skonfiguruj go według własnych preferencji.

[edytuj] Konfiguracja

[edytuj] Przykład prostego pliku konfiguracyjnego

File: /boot/syslinux/syslinux.cfg
PROMPT 1
TIMEOUT 50
DEFAULT arch

LABEL arch
        LINUX ../vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD ../kernel26.img

LABEL archfallback
        LINUX ../vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD ../kernel26-fallback.img

[edytuj] Utworzenie tekstowego menu startowego

Syslinux także umożliwia korzystanie z tekstowego menu startowego. Aby go użyć, skopiuj moduł COM32 menu do folderu /boot/syslinux:

# cp /usr/lib/syslinux/menu.c32 /boot/syslinux/

Nastepnie dodaj przykładowe wiersze:

File: /boot/syslinux/syslinux.cfg
UI menu.c32
PROMPT 0

MENU TITLE Boot Menu
TIMEOUT 50
DEFAULT arch

LABEL arch
        MENU LABEL Arch Linux
        LINUX ../vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD ../kernel26.img

LABEL archfallback
        MENU LABEL Arch Linux Fallback
        LINUX /vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD /kernel26-fallback.img

[edytuj] Utworzenie graficznego menu startowego

Aby posiadać graficzne menu startowe, skopiuj moduł COM32 menu do folderu /boot/syslinux:

# cp /usr/lib/syslinux/vesamenu.c32 /boot/syslinux/

Należy zaznaczyć, że jest tu wykorzystany wygląd menu z płyty instalacyjnej Archa (syslinux.cfg). Tło można znaleźć tu.

File: /boot/syslinux/syslinux.cfg
UI vesamenu.c32
DEFAULT arch
PROMPT 0
MENU TITLE Boot Menu
MENU BACKGROUND splash.png
TIMEOUT 50

MENU WIDTH 78
MENU MARGIN 4
MENU ROWS 5
MENU VSHIFT 10
MENU TIMEOUTROW 13
MENU TABMSGROW 11
MENU CMDLINEROW 11
MENU HELPMSGROW 16
MENU HELPMSGENDROW 29

# Refer to http://syslinux.zytor.com/wiki/index.php/Doc/menu

MENU COLOR border       30;44   #40ffffff #a0000000 std
MENU COLOR title        1;36;44 #9033ccff #a0000000 std
MENU COLOR sel          7;37;40 #e0ffffff #20ffffff all
MENU COLOR unsel        37;44   #50ffffff #a0000000 std
MENU COLOR help         37;40   #c0ffffff #a0000000 std
MENU COLOR timeout_msg  37;40   #80ffffff #00000000 std
MENU COLOR timeout      1;37;40 #c0ffffff #00000000 std
MENU COLOR msg07        37;40   #90ffffff #a0000000 std
MENU COLOR tabmsg       31;40   #30ffffff #00000000 std


LABEL arch
        MENU LABEL Arch Linux
        LINUX ../vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD ../kernel26.img


LABEL archfallback
        MENU LABEL Arch Linux Fallback
        LINUX ../vmlinuz26
        APPEND root=/dev/sda2 ro
        INITRD ../kernel26-fallback.img

[edytuj] Dodatkowe wpisy w menu

Do menu startowego można dodać następujące opcje:

  • Memtest86+ 
LABEL memtest
MENU LABEL Memtest86+
LINUX /memtest86+/memtest.bin
  • HDT (Hardware Detection Tool) 
LABEL hdt
MENU LABEL Hardware Info
COM32 hdt.c32
