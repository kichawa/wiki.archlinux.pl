[edytuj] Wprowadzenie

The Movie Player powstał w 2000 roku, jako odtwarzacz filmów dla systemu operacyjnego Linux. Jego ogromna popularność oraz porty na kolejne platformy sprzętowe i systemy operacyjne spowodowały w 2004 roku zmianę nazwy z The Movie Player for Linux na The Movie Player.

[edytuj] Instalacja

# pacman -S mplayer

[edytuj] Kodeki

Są dwa warianty instalacji kodeków. Pierwszym jest pobranie kodeków zamieszczonych na stronie MPlayerHQ. Pakiet zawierający wszystkie kodeki dostępny jest w Arch User Repository.

Drugim wariantem są kodeki GStreamer:

# pacman -S gstreamer0.10-{{bad,good,ugly,base}{,-plugins},ffmpeg}

[edytuj] Nakładki graficzne

  • SMPlayer
# pacman -S smplayer smplayer-themes
  • KMPlayer
# pacman -S kmplayer
  • GNOME-MPlayer
# pacman -S gnome-mplayer

[edytuj] Konfiguracja

Plik konfiguracyjny znajdziesz w /etc/mplayer/mplayer.conf. Jest to jesdnak konfiguracja globalna, dla wszystkich użytkowników systemu. Jeżeli chcesz skonfigurować MPlayera dla wybranego użytkownika, należy plik ~/.mplayer/config i w nim nanieść swoje ustawienia.

Przykłady plików konfiguracyjnych:

File: /etc/mplayer/example.conf
# default configuration that applies to every file
[default]
# use X11 for video output
vo=x11
# use also for audio output
ao=alsa
# ao=oss # Use OSS4
# prefer using six channels audio
channels = 6
# scale the subtitles to the 3% of the screen size
subfont-text-scale = 3
# never use font config
nofontconfig = 1
# add black borders so the movies have the same aspect ratio of the monitor
# for wide screen monitors
vf-add=expand=::::1:16/9:16
# for non wide screen traditional monitors
#vf-add=expand=::::1:4/3:16

#profile for up-mixing two channels audio to six channels
# use -profile 2chto6ch to activate
[2chto6ch]
af-add=pan=6:1:0:.4:0:.6:2:0:1:0:.4:.6:2
 
#profile to down-mixing six channels audio to two channels
# use -profile 6chto2ch to activate
[6chto2ch]
af-add=pan=2:0.7:0:0:0.7:0.5:0:0:0.5:0.6:0.6:0:0

# Disable screensaver.
heartbeat-cmd="xscreensaver-command -deactivate &" # stop xscreensaver
stop-xscreensaver="yes" # stop gnome-screensaver
File: ~/.mplayer/config
[default]

#=======================
# Ogolne
#=======================
# Uruchamianie filmu na pelnym ekranie
# fs=yes
# Wylaczenie wygaszacza ekranu na czas odtwarzania
stop-xscreensaver=yes
# Wylaczenie czarnych obramowan
#vf-add=expand=::::1:16/9:16
#=======================
# Wideo
#=======================
# Sterownik obrazu
# vo=gl
vo=vaapi:gl
# Buforowanie
# double=yes
# Direct Rendering 
# dr=yes
# W niektórych filmach może nie działać gamma, itp. (zdarza się na pewno na vo=xv)
#vf-add=eq2
# Domyślnie panscan=0
#panscan=0.5
#=======================
# Audio
#=======================
# Język
alang=pl,en
# Sterownik dzwieku
ao=alsa
# Mikser
mixer-channel=Master
# Programowy mixer
#softvol=1
# Wzmocnienie (domyślnie będzie 100% ale ale skala będzie do 300%)
#softvol-max=300
# Synchronizacja dzwieku
# mc=0.2
# autosync=30
#=======================
# Napisy
#=======================
# Jezyk
slang=pl
# Kodowanie
subcp=cp1250
subfont-encoding=unicode
unicode=yes
utf8=yes
# Wczytywanie napisów o tej samej nazwie
sub-fuzziness=1
# Czcionka
font=/usr/share/fonts/TTF/arial.ttf
# Wielkosc czcionki
subfont-text-scale=3
subfont-autoscale=3
# Czarna otoczka napisow
ffactor=10
# Antialiasing czcionki
spuaa=4
# Dodatkowe:
#fontconfig=yes
#osd-duration=2000
#subfont-outline=2
#subfont-blur=4
#subfont-osd-scale=2,2
# ALTERNATYWNE: SSA/ASS (to + panscan = ucięte litery)
#vf-add=ass
#ass=yes
#embeddedfonts=yes
#ass-use-margins=yes
#ass-color=FFFFFF00 # RRGGBBAA - plain white (no alpha)
#ass-border-color=00000000
#ass-font-scale=1.6
#ass-hinting=1
#=======================
# Wydajnosc
#=======================
# autoq=100
# vf-add=pp=de,hqdn3d
# cache=2080
#=======================
# Inne
#=======================
# Bez joysticka, bez pilota
#nojoystick=yes
#nolirc=yes
# Kolorowe wyjście na konsoli (polepsza czytelność)
#msgcolor=yes
# Menu (Trzeba mieć mplayera skompilowanego z tą funkcją)
#menu=yes
# Menu czuje ruch kursorem myszki
#mouse-movements=yes

[edytuj] Menu OSD

Aktualizujesz drzewo ABS albo używasz pbget. Następnie edytujesz plik PKGBUILD i zmieniasz linijkę: 

--disable-menu \

na: 

--enable-menu \

i kompilujesz (makepkg -i)..

Menu które używam:

File: ~/.mplayer/menu.conf
<keybindings name="default">
    <binding key="UP" cmd="menu up" />
    <binding key="DOWN" cmd="menu down" />
    <binding key="LEFT" cmd="menu left" />
    <binding key="RIGHT" cmd="menu right" />
    <binding key="ENTER" cmd="menu ok" />
    <binding key="ESC" cmd="menu cancel" />
    <binding key="HOME" cmd="menu home" />
    <binding key="END" cmd="menu end" />
    <binding key="PGUP" cmd="menu pageup" />
    <binding key="PGDWN" cmd="menu pagedown" />
    <binding key="MOUSE_BTN0" cmd="menu ok" />
    <binding key="MOUSE_BTN1" cmd="menu hide" />
    <binding key="MOUSE_BTN2" cmd="menu cancel" />
    <binding key="MOUSE_BTN3" cmd="menu up" />
    <binding key="MOUSE_BTN4" cmd="menu down" />
</keybindings>

<console name="console0" height="80" vspace="0">Witamy w MPlayer</console>

<pt name="jump_to" title="Playlista:" />
<chapsel name="select_chapter" title="Wybierz rozdział:" />

<pref name="general_pref" title="Ogólne">
   <e property="osdlevel" name="Poziom OSD"/>
   <e property="speed" name="Prędkość"/>
   <e property="loop" name="Pętla"/>
</pref>

<pref name="demuxer_pref" title="Demuxer">
</pref>

<pref name="osd_sub_pref" title="Napisy">
   <e property="sub" name="Napisy"/>
   <e property="sub_visibility" name="Wl/Wył napisy"/>
   <e property="sub_forced_only" name="Forced sub only"/>
   <e property="sub_alignment" name="Wyrownanie"/>   
   <e property="sub_pos" name="Pozycja"/>
   <e property="sub_delay" name="Opóźnienie"/>
   <e property="sub_scale" name="Skala"/>
</pref>

<pref name="audio_pref" title="Dźwięk">
   <e property="switch_audio" name="Zmień ścieżkę dźwiękową"/>
   <e property="volume" name="Głośność"/>
   <e property="balance" name="Balans"/>
   <e property="mute" name="Wyciszenie"/>
   <e property="audio_delay" name="Opóźnienie"/>
</pref>

<pref name="colors_pref" title="Kolory">
   <e property="gamma" name="Gamma"/>
   <e property="brightness" name="Ostrość"/>
   <e property="contrast" name="Kontrast"/>
   <e property="saturation" name="Nasycenie"/>
   <e property="hue" name="Hue"/>
</pref>

<pref name="video_pref" title="Obraz">
   <e property="fullscreen" name="Pełen ekran"/>
   <e property="panscan" name="Panscan"/>
   <menu menu="colors_pref" name="Kolory.."/>
   <e property="framedropping" name="Zrzucanie ramek"/>
   <e property="ontop" name="Zawsze na wierzchu"/>
   <e property="vsync" name="VSync"/>
</pref>

<cmdlist name="pref_main" title="Ustawienia">
   <e name="Ogólne.." ok="set_menu general_pref"/>
   <e name="Dźwięk.." ok="set_menu audio_pref"/>
   <e name="Obraz.." ok="set_menu video_pref"/>
   <e name="Napisy.." ok="set_menu osd_sub_pref"/>
   <e name="Back" ok="menu cancel"/>
</cmdlist>

<pref name="properties" title="Informacje o pliku">
   <e txt="${filename}" name="Nazwa"/>
   <e txt="${video_format}" name="Kodek wideo"/>
   <e txt="${video_bitrate}" name="Wideo bitrate"/>

   <e txt="${audio_bitrate}" name="Audio bitrate"/>
   <e txt="${samplerate}, ${channels}" name="Audio rate"/>
   <e txt="${metadata/Title}" name="Tytuł"/>
   <e txt="${metadata/Artist}" name="Artysta"/>
   <e txt="${metadata/Album}" name="Album"/>
   <e txt="${metadata/Year}" name="Rok"/>
   <e txt="${metadata/Comment}" name="Komentarz"/>
   <e txt="${metadata/Track}" name="Ścieżka"/>
   <e txt="${metadata/Genre}" name="Gatunek"/>
   <e txt="${metadata/Software}" name="Oprogramowanie"/>
</pref>

<cmdlist name="main" title="MPlayer OSD menu">
   <e name="Pauza" ok="pause"/>
   <e name="Playlista.." ok="set_menu jump_to" left="pt_step -1" right="pt_step +1"/>
   <e name="Rozdziały DVD.." ok="set_menu select_chapter" left="seek_chapter -1" right="seek_chapter +1"/>
   <e name="Ustawienia.." ok="set_menu pref_main"/>
   <e name="Informacje" ok="set_menu properties"/>
   <e name="Wyjdź" ok="quit"/>
</cmdlist>

Alternatywny plik menu.conf z projektu mplayer-ce (wersja: svn-trunk).

Już tylko zostało upewnić się że mamy ustawiony skrót klawiszowy dla menu. W pliku ~/.mplayer/input.conf powinna znajdować się taka linijka: 

\ menu up

gdzie w tym przypadku "\" jest skrótem, który wyświetli menu. Jeżeli nie masz tego pliku, skopiuj go z /etc/mplayer/input.conf we wcześniej podane miejsce.

[edytuj] Uruchamianie plików ze spacją w nazwie

Uruchamianie plików ze spacją w nazwie jest niemożliwe bez wprowadzania kilku zmian. Aby wszystko działało, edytuj aktywator MPlayera (/usr/share/applications/mplayer.desktop) i zmień z: 

Exec=mplayer %U

na 

Exec=mplayer %F

[edytuj] MPlayer i PulseAudio

Aby Mplayer korzystał z PulseAudio, należy w jego pliku konfiguracyjnym dodać/zmienić poniższy wiersz:

File: ~/.mplayer/config
ao=pulse

[edytuj] Skróty klawiaturowe

Poniższa lista przedstawia tylko najczęściej używane skróty. Aby dowiedzieć się więcej, odwiedź stronę. 

p 	      Pauza/Wznowienie
Space 	      Pauza/Wznowienie
Left 	      Cofnij czas odtwarzania filmu o 10 sekund
Right 	      Dodaj do czasu odtwarzania filmu 10 sekund
Down 	      Cofnij czas odtwarzania filmu o minutę
Up 	      Dodaj minutę do czasu odtwarzania filmu
m 	      Wyłącz dźwięk
0 	      Zwiększ poziom dźwięku o jeden poziom
9 	      Zmniejsz poziom dźwięku o jeden poziom
f 	      Przełączanie trybu pełnoekranowego
1, 2 	      Kontrola kontrastu
3, 4 	      Kontrola jasności
v             Przełączanie wyświetlania napisów
b, j          Przełączanie dostępnych napisów
z, x          Zmienianie opóźnienia napisów o +/- 0.1 sekund
r, t          Przesuwanie napisów w górę/w dół
