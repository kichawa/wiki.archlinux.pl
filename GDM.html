[edytuj] Wprowadzenie

GDM (GNOME Display Manager) to menedżer logowania dla środowiska graficznego GNOME. Został zaprojektowany jako prostsza w konfiguracji alternatywa dla domyślnego w X11 menedżera XDM.

[edytuj] Instalacja

# pacman -S gdm

a następnie na końcu sekcji DAEMONS w pliku /etc/rc.conf dodajemy odpowiedni wpis:

File: /etc/rc.conf
DAEMONS=(...gdm)

Ręczne uruchomienie:

# /etc/rc.d/gdm start

Istnieje też druga metoda uruchamiania GDM (bez udziału /etc/rc.conf). Edytujemy /etc/inittab i ustawiamy tak, aby wyglądało to tak jak na poniższym przykładzie:

File: /etc/inittab
...
# Boot to console                                         
#id:3:initdefault:                                        
# Boot to X11                                             
id:5:initdefault:
...
# Example lines for starting a login manager
#x:5:respawn:/usr/bin/xdm -nodaemon
x:5:respawn:/usr/sbin/gdm -nodaemon
#x:5:respawn:/usr/bin/kdm -nodaemon
#x:5:respawn:/usr/bin/slim >& /dev/null
#x:5:once:/bin/su nazwa_użytkownika -l -c "/bin/bash --login -c startx >/dev/null 2>&1"
#x:5:respawn:/usr/sbin/entranced --nodaemon >& /dev/null

Do pliku ~/.xinitrc znajdującego się w katalogu domowym należy dodać:

File: /etc/~/.xinitrc
exec gnome-session

i wszystkie inne linijki zaczynające się od exec zakomentować, a następnie uruchomić:

$ startx

[edytuj] Konfiguracja GDM

[edytuj] Autologowanie użytkownika

Aby dany użytkownik logował się automatycznie, należy dodać poniższy wpis do /etc/gdm/custom.conf:

File: /etc/gdm/custom.conf
[daemon]
AutomaticLoginEnable=true
AutomaticLogin=nazwa_użytkownika

[edytuj] Logowanie na konto root

Jeżeli chcesz zalogować się na konto root, dodaj poniższy wpis do pliku /etc/gdm/custom.conf:

File: /etc/gdm/custom.conf
[security]
AllowRoot=true

[edytuj] Ustawienia polskie w oknie GDM

Aby mieć w oknie GDM ustawienia polskie, należy wyedytować plik ~/.drmc, aby wyglądał jak na poniższym przykładzie:

File: ~/.drmc
[Desktop]
Language=pl_PL.UTF-8
Layout=pl

Drugim sposobem jest utworzenie pliku /etc/X11/xorg.conf.d/10-keyboard.conf z zawartością:

File: /etc/X11/xorg.conf.d/10-keyboard.conf
Section "InputClass"
Identifier              "Polish Keyboard"
MatchIsKeyboard         "yes"
Option   "XkbLayout"    "pl"
Option   "XkbVariant"   "nodeadkeys"
EndSection

[edytuj] Nowy sposób konfiguracji w GNOME 3

Na początek wydaj polecenia:

# su - gdm -s /bin/bash
$ dbus-launch

Otrzymasz wartość DBUS_SESSION_BUS_ADDRESS oraz DBUS_SESSION_BUS_PID. Wpisz je w konsoli, na przykład:

$ export DBUS_SESSION_BUS_ADDRESS=unix:abstract=/tmp/dbus-Jb433gMQHS,guid=fc14d4bf3d000e38276a5a2200000d38
$ export DBUS_SESSION_BUS_PID=4283

Sprawdź, czy uruchomniony jest dconf-service. Jeśli nie, uruchom go:

$ /usr/lib/dconf/dconf-service &

Teraz przyszedł czas na konfigurację.

Aby zmienić tapetę, wpisz:

$ GSETTINGS_BACKEND=dconf gsettings get org.gnome.desktop.background picture-uri
$ GSETTINGS_BACKEND=dconf gsettings set org.gnome.desktop.background picture-uri "file:///usr/share/backgrounds/gnome/SundownDunes.jpg"

Aby wyłączyć dźwięk:

$ GSETTINGS_BACKEND=dconf gsettings set org.gnome.desktop.sound event-sounds false

[edytuj] Konfiguracja GDM dla GNOME do wersji 2.32

Ostrzeżenie: Poniższe informacje nie są przeznaczone dla GNOME 3.

[edytuj] Zmiana wyglądu GDM - Metoda 1

Aby zmienić wygląd domyślnego motywu GDM, wystarczy wpisać polecenie:

  • w przypadku zainstalowanego sudo
# gksudo -u gdm dbus-launch gnome-appearance-properties
  • w przypadku, gdy nie używamy sudo
# gksu -u gdm dbus-launch gnome-appearance-properties

[edytuj] Zmiana wyglądu GDM - Metoda 2

Otwieramy gconf-editor:

# -u gdm dbus-launch gconf-editor

Ustawienia GDM:

  • Tapeta 
/desktop/gnome/background/picture_filename

Ewentualnie możemy podmienić plik, zachowując nazwę oryginalnego, w katalogu /usr/share/pixmaps/backgrounds/gnome.

  • Motyw GTK 
/desktop/gnome/interface/gtk_theme
  • Motyw ikon 
/desktop/gnome/interface/icone_theme
  • Ikona w oknie logowania 
/apps/gdm/simple-greeter/logo_icon_name

[edytuj] Zmiana wyglądu GDM - Metoda 3

Ustawienia GDM:

  • Tapeta

Polecenie wyświetli nam obecną tapetę

# -u gdm gconftool-2 --get /desktop/gnome/background/picture_filename

Polecenie ustawi nam nową tapetę

# -u gdm gconftool-2 --set --type string /desktop/gnome/background/picture_filename /data/pictures/Grassy.jpg

Ewentualnie możemy podmienić plik, zachowując nazwę oryginalnego, w katalogu /usr/share/pixmaps/backgrounds/gnome.

  • Motyw GTK

Polecenie wyświetli nam obecny motyw GTK

# -u gdm gconftool-2 --get /desktop/gnome/interface/gtk_theme

Polecenie ustawi nam nowy motyw GTK

# -u gdm gconftool-2 --set --type string /desktop/gnome/interface/gtk_theme Redmond
  • Motyw ikon

Polecenie wyświetli nam obecny motyw ikon

# -u gdm gconftool-2 --get /desktop/gnome/interface/icon_theme

Polecenie ustawi nam nowy motyw ikon

# -u gdm gconftool-2 --set --type string /desktop/gnome/interface/icon_theme Tangerine
  • Ikona w oknie logowania

Polecenie wyświetli nam obecną ikonę

# -u gdm gconftool-2 --get /apps/gdm/simple-greeter/logo_icon_name

Polecenie ustawi nam nową ikonę

# -u gdm gconftool-2 --set --type string /apps/gdm/simple-greeter/logo_icon_name gnome-globe

[edytuj] Konfiguracja wyświetlania kont

Otwieramy gconf-editor:

# -u gdm dbus-launch gconf-editor

Ustawienia znajdziemy w: 

/apps/gdm/simple-greeter/disable_user_list

Wyświetlanie kont możemy wyłączyć także za pomocą polecenia:

# -u gdm gconftool-2 --set --type boolean /apps/gdm/simple-greeter/disable_user_list true

Natomiast poniższe polecenie włącza wyświetlanie kont w GDM:

# -u gdm gconftool-2 --set --type boolean /apps/gdm/simple-greeter/disable_user_list false

[edytuj] GDM i Compiz

Otwieramy gconf-editor:

# -u gdm dbus-launch gconf-editor

Ustawienia znajdziemy w: 

/apps/gdm/simple-greeter/wm_use_compiz

Menedżera Compiz możemy wyłączyć także za pomocą polecenia:

# -u gdm gconftool-2 --set --type boolean /apps/gdm/simple-greeter/wm_use_compiz false

Natomiast poniższe polecenie włącza menedżera Compiz w GDM:

# -u gdm gconftool-2 --set --type boolean /apps/gdm/simple-greeter/wm_use_compiz true
