[edytuj] Wprowadzenie

Laptop Mode Tools jest narzędziem oferującym proste i wygodne zarządzanie energią w komputerach przenośnych. Oprogramowanie daje użytkownikowi możliwość oszczędzania energii podczas pracy na baterii. Pozwala sterować opcjami uśpienia systemu czy częstotliwością taktowania procesora.

[edytuj] Instalacja

# pacman -S laptop-mode-tools

[edytuj] Konfiguracja

  • Dodaj laptop-mode do tablicy DAEMONS w pliku /etc/rc.conf.
  • Pliki konfiguracyjne można znaleźć w pliku /etc/laptop-mode/laptop-mode.conf i /etc/laptop-mode/conf.d/*.
  • Pamiętaj, aby przejść przez pliki konfiguracyjne, jako że wiele ciekawych funkcji jest domyślnie wyłączonych.

[edytuj] Konfiguracja dysku twardego

Do zrobienia: Opisać konfigurację ujętą w pliku /etc/laptop-mode/laptop-mode.conf.

[edytuj] Konfiguracja CPU

File: /etc/laptop-mode/conf.d/cpufreq.conf
# ThinkPad T40/T42/T60 Example
#
CONTROL_CPU_FREQUENCY=1
BATT_CPU_MAXFREQ=fastest
BATT_CPU_MINFREQ=slowest
BATT_CPU_GOVERNOR=ondemand
BATT_CPU_IGNORE_NICE_LOAD=1
LM_AC_CPU_MAXFREQ=fastest
LM_AC_CPU_MINFREQ=slowest
LM_AC_CPU_GOVERNOR=ondemand
LM_AC_CPU_IGNORE_NICE_LOAD=1
NOLM_AC_CPU_MAXFREQ=fastest
NOLM_AC_CPU_MINFREQ=slowest
NOLM_AC_CPU_GOVERNOR=ondemand
NOLM_AC_CPU_IGNORE_NICE_LOAD=0
CONTROL_CPU_THROTTLING=0

[edytuj] Konfiguracja usypiania USB

File: /etc/laptop-mode/conf.d/usb-autosuspend.conf
# ThinkPad T40/T42/T60 Example
#
DEBUG=0
CONTROL_USB_AUTOSUSPEND=1
BATT_SUSPEND_USB=1
LM_AC_SUSPEND_USB=1
NOLM_AC_SUSPEND_USB=0
AUTOSUSPEND_TIMEOUT=2

[edytuj] Konfiguracja jasności ekranu LCD

Dostępne wartości jasności możesz uzyskać dzięki poleceniu:

$ cat /proc/acpi/video/VID/LCD/brightness

Lub

$ cat /sys/class/backlight/acpi_video0/brightness
File: /etc/laptop-mode/conf.d/lcd-brightness.conf
# ThinkPad T40/T42 Example
#
DEBUG=0
CONTROL_BRIGHTNESS=1
BATT_BRIGHTNESS_COMMAND="echo 0"
LM_AC_BRIGHTNESS_COMMAND="echo 7"
NOLM_AC_BRIGHTNESS_COMMAND="echo 7"
BRIGHTNESS_OUTPUT="/sys/class/backlight/thinkpad_screen/brightness"
File: /etc/laptop-mode/conf.d/lcd-brightness.conf
# ThinkPad T60 Example
#
DEBUG=0
CONTROL_BRIGHTNESS=1
BATT_BRIGHTNESS_COMMAND="echo 0"
LM_AC_BRIGHTNESS_COMMAND="echo 7"
NOLM_AC_BRIGHTNESS_COMMAND="echo 7"
BRIGHTNESS_OUTPUT="/sys/class/backlight/acpi_video0/brightness"

[edytuj] Konfiguracja wygaszania terminala

File: /etc/laptop-mode/conf.d/terminal-blanking.conf
# ThinkPad T40/T42/T60 Example
#
DEBUG=0
CONTROL_TERMINAL=1
TERMINALS="/dev/tty1"
BATT_TERMINAL_BLANK_MINUTES=1
BATT_TERMINAL_POWERDOWN_MINUTES=2
LM_AC_TERMINAL_BLANK_MINUTES=10
LM_AC_TERMINAL_POWERDOWN_MINUTES=10
NOLM_AC_TERMINAL_BLANK_MINUTES=10
NOLM_AC_TERMINAL_POWERDOWN_MINUTES=10

[edytuj] Konfiguracja sieci

File: /etc/laptop-mode/conf.d/ethernet.conf
# ThinkPad T40/T42/T60 Example
#
DEBUG=0
CONTROL_ETHERNET=1
LM_AC_THROTTLE_ETHERNET=0
NOLM_AC_THROTTLE_ETHERNET=0
DISABLE_WAKEUP_ON_LAN=1
DISABLE_ETHERNET_ON_BATTERY=1
ETHERNET_DEVICES="eth0"

Konfiguracja Wifi

Ustawienia interfejs zarządzania sieci bezprzewodowej zależą od sprzętu, są nieco trudniejsze w konfiguracji. W zależności od chipsetu, ustawienia są zarządzane w jednym z trzech plików:

  1. /etc/laptop-mode/conf.d/wireless-power.conf - ogólne zasady oszczędzania energii.
  2. /etc/laptop-mode/conf.d/wireless-ipw-power.conf - dla chipsetów Intel i starego sterownika ipw.
  3. /etc/laptop-mode/conf.d/wireless-iwl-power.conf - dla chipsetów Intel z modułami iwl4965, iwl3945 i iwlagn.

[edytuj] Konfiguracja dźwięku

AC97

File: /etc/laptop-mode/conf.d/ac97-powersave.conf
# ThinkPad T40/T42/T60 Example
#
DEBUG=0
CONTROL_AC97_POWER=1

Intel HDA

File: /etc/laptop-mode/conf.d/intel-hda-powersave.conf
# ThinkPad T40/T42/T60 Example
#
DEBUG=0
CONTROL_INTEL_HDA_POWER=1
BATT_INTEL_HDA_POWERSAVE=1
LM_AC_INTEL_HDA_POWERSAVE=1
NOLM_AC_INTEL_HDA_POWERSAVE=0
INTEL_HDA_DEVICE_TIMEOUT=10
INTEL_HDA_DEVICE_CONTROLLER=0
