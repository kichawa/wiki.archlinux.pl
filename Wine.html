[edytuj] Wprowadzenie

Wine [Wine Is Not an Emulator] jest oprogramowaniem umożliwiającym wykonywanie aplikacji przeznaczonych dla Microsoft Windows w środowisku systemu operacyjnego klasy Unix [Linux, Solaris, FreeBSD]. Przy pomocy bibliotek Wine można również przenosić niektóre aplikacje systemu Windows do natywnych plików binarnych systemu Unix.

[edytuj] Instalacja 

# pacman -S wine

W przypadku architektury x86_64 zalecana jest instalacja pakietów lib32-nvidia-utils dla kart Nvidia i lib32-catalyst-utils dla kart ATI.

[edytuj] Konfiguracja

Po instalacji należy uruchomić konfigurator i skonfigurować Wine: 

# winecfg

Bardzo możliwe, że po instalacji nie będzie w naszym Menu widocznego Wine. Oto trzy aktywatory, które pomogą w zarządzaniu: 

Typ: Aplikacja 
Nazwa: Konfiguracja 
Polecenie: winecfg 

Typ: Aplikacja 
Nazwa: Odinstaluj programy 
Polecenie: wine uninstaller 

Typ: Aplikacja 
Nazwa: Przeglądaj C:\
Polecenie: winebrowser wina c:\\

Jeżeli po uruchomieniu jakiegoś programu nie widzisz tekstu, oznacza to, że nie masz zainstalowanych fontów TTF Microsoftu: 

# pacman -S ttf-ms-fonts

[edytuj] Uruchamianie programów

Aby uruchomić windowsowy program za pomocą Wine pod GNU/Linuksem wydajemy polecenie: 

$ wine /tu/ścieżka/do/programu/program.exe

Jeżeli chcesz uruchomić program z zaznaczoną architektura Wine: 

$ WINEARCH=win32 wine /tu/ścieżka/do/programu/program.exe

WINEARCH=win32 oznacza 32-bitowe Wine. Natomiast, jeżeli posiadasz kilka konfiguracji Wine, musisz wpisać podobne polecenie: 

$ WINEPREFIX=/ścieżka/do/konfiguracji/wine wine /tu/ścieżka/do/programu/program.exe

[edytuj] Winegame

Winegame to skrypt umożliwiający instalację różnych, ogólnie dostępnych bibliotek systemowych, które niekiedy są potrzebne do uruchomienia programów w Wine. 

# pacman -S winegame

[edytuj] Winedoors

Wine-doors to aplikacja stworzona po to, by instalacja windowsowych programów na Linuksie, Solarisie i innych systemach z rodziny Unix była jeszcze łatwiejsza. Program działa podobnie do menedżera pakietów znanego większości użytkownikom Linuksa. Wystarczy wybrać z listy interesującą nas aplikację, a zostanie ona pobrana i zainstalowana na naszym komputerze. Pakiet dostępny jest w Arch User Repository.

[edytuj] Winetricks

Winetricks to skrypt powłoki umożliwiający instalację różnych, ogólnie dostępnych bibliotek systemowych, które niekiedy są potrzebne do uruchomienia programów w Wine. 

# pacman -S winetricks

[edytuj] PlayOnLinux

PlayOnLinux to swego rodzaju nakładka na Wine, która pozwala w dość łatwy sposób zainstalować gry i programy z Windows na Linuksie, a także je konfigurować, czy aktualizować. Zainstalować można wiele innych gier dzięki dodatkowym skryptom pisanym przez społeczność. Pakiet dostępny jest w Arch User Repository.

[edytuj] Vineyard

Vineyard jest narzędziem, które może zastąpić domyślny konfigurator Wine. Zostało stworzone, aby ułatwić pracę z Wine i zapewnić bardzo dobrą integrację ze środowiskiem GNOME. Pakiet dostępny jest w Arch User Repository.
