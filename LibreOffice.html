[edytuj] Wprowadzenie

LibreOffice jest pakietem biurowym, tworzonym przez część deweloperów OpenOffice.org (zakończyli współpracę z Oracle) pod nadzorem niezależnej organizacji The Document Foundation.

Jeden z "ojców" nowego pakietu biurowego, Charles H. Schulz, powiedział, że nie uważa LibreOffice za fork OpenOffice.org, lecz raczej kontynuację oryginalnego projektu.

[edytuj] Instalacja

# pacman -S libreoffice libreoffice-pl

W Arch User Repository znajdziesz wersję Beta.

Jeżeli chciałbyś rozszerzyć mozliwości LibreOffice, zainstaluj grupę libreoffice-extensions lub pobierz rozszerzenia ze strony LibrePlanet.

[edytuj] Instalacja polskiego słownika

Ściągamy zestaw polskich słowników. Następnie dodajemy go w zakładce Narzędzia -> Menedżer rozszerzeń.

[edytuj] Zalecane pakiety

  • Należy zainstalować Java Runtime Environment (opcjonalne).
# pacman -S openjdk6

Lub

# pacman -S jre
  • Czionki
# pacman -S ttf-dejavu ttf-liberation

[edytuj] Optymalizacja LibreOffice

  • Narzędzia > Opcje > Pamięć:
Zmniejsz liczbę kroków na wartość 20-30.
Pamięć podręczną na grafikę zwiększ do 128 MB.
Ilość pamięci na obiekt zwiększ do 20 MB.
  • Narzędzia > Opcje > Java:
Odznacz "Użyj środowiska Java".

[edytuj] Uruchamianie programów pakietu LibreOffice

Writer 

/usr/bin/libreoffice -writer
/usr/bin/soffice -writer

Calc 

/usr/bin/libreoffice -calc

Impress 

/usr/bin/libreoffice -impress

Draw 

/usr/bin/libreoffice -draw

Math 

/usr/bin/libreoffice -math

Base 

/usr/bin/libreoffice -base
