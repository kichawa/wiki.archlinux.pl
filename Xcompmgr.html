[edytuj] Wprowadzenie

Xcompmgr jest prostym menedżerem okien, wykorzystywanym na przykład w Fluxboksie czy Openboksie.

[edytuj] Zanim zaczniesz

Xcompmgr wymaga:

  • zainstalowanego i skonfigurowanego serwera X;
  • zainstalowanych i skonfigurowanych sterowników do twojej karty graficznej.

[edytuj] Instalacja 

# pacman -S xcompmgr transset

Możesz zainstalować wersję rozszerzoną - xcompmgr-dana.

[edytuj] Konfiguracja

Aby uruchomić Xcompmgr, wpisz: 

$ xcompmgr -c

Aby narzędzie ładowało się z każdym uruchomieniem systemu, dodaj do autostartu: 

xcompmgr -c &

Zamiast -c możesz poeksperymentować z różnymi opcjami dodatkowymi, na przykład: 

xcompmgr -c -t-5 -l-5 -r4.2 -o.55 &

Pełną listę opcji uzyskasz wpisując: 

$ xcompmgr --help

[edytuj] Przezroczystość okien

Mimo, że praktyczne wykorzystanie tej funkcji jest ograniczone ze względu na niską wydajność, użyteczność transset może być użyta do ustawienia przejrzystości poszczególnych okien.

Aby ustawić przezroczystość okna programu, upewnij się, że żądany program jest już uruchomiony, a następnie wykonaj: 

$ transset n

Gdzie n oznacza liczbę od 0 do 1 (zero - przejrzysty, 1 - nieprzejrzysty).

Po wykonaniu powyższego polecenia kursor zmieni się w krzyżyk. Wystarczy teraz kliknąć na żądane okno i przejrzystość jego zmieni się na określoną wcześniej wartość.

Dla przykładu, polecenietransset .25 ustawi przezroczystość okna na 75%.

[edytuj] Rozwiązywanie problemów

[edytuj] Przeglądarka Firefox nie działa po wejściu na stronę z animacjami Flash

Możesz to naprawić poprzez stworzenie pliku wykonywalnego /etc/profile.d/flash.sh, zawierającego poniższą linię:

File: /etc/profile.d/flash.sh
export XLIB_SKIP_ARGB_VISUALS=1

[edytuj] Po zalogowaniu tło zmienia się na kolor jasnoszary

  • Zainstaluj hsetroot
  • Użyj polecenia hsetroot -solid "#000000" (możesz wpisać dowolny kolor) w autostarcie. Polecenie powinno uruchamiać się przed Xcompmgr.

[edytuj] Porady

[edytuj] Uruchamianie/Restart/Zatrzymywanie Xcompmgr na żądanie

Ten skrypt pozwoli tobie na uruchomienie, restart czy też wyłączenie Xcompmgr: 

#!/bin/bash
#
# Start a composition manager.
# (xcompmgr in this case)

function comphelp () {
  echo "Composition Manager:"
  echo "   (re)start: COMP"
  echo "   stop:      COMP -s"
  echo "   query:     COMP -q"
  echo "              returns 1 if composition manager is running, else 0"
  exit
}

function checkcomp () {
  ( ps nc -C xcompmgr &>/dev/null ) && exit 1
  exit 0
}

function stopcomp () {
  ( ps nc -C xcompmgr &>/dev/null ) && killall xcompmgr
}

function startcomp () {
  stopcomp
# Example settings only. Replace with your own.
  xcompmgr -CcfF -I-.015 -O-.03 -D6 -t-1 -l-3 -r4.2 -o.5 &
  exit
}

case "$1x" in
    "x") startcomp;;
  "-qx") checkcomp;;
  "-sx") stopcomp; exit;;
      *) comphelp;;
esac
