[edytuj] Wprowadzenie

W tym artykule znajdziesz informacje, jak spolszczyć swój system. Przede wszystkim obowiązkowo wykonaj polecenia w punkcie "Podstawowe ustawienia". Konfiguracja Xorg jest zalecana. Nastepnie sprawdź konfigurację językową dla twojego środowiska graficznego. To tyle.

[edytuj] Podstawowe ustawienia

Najpierw edytujemy plik /etc/rc.conf, aby wyglądał tak:

File: /etc/rc.conf
LOCALE="pl_PL.UTF-8"          # Ewentualnie LOCALE="pl_PL.ISO-8859-2"
HARDWARECLOCK="localtime"     # UTC - linux, localtime - linux & Windows
TIMEZONE="Europe/Warsaw"
KEYMAP="pl"
CONSOLEFONT="lat2-16.psfu.gz" # W tym miejscu możesz zmienić czcionkę na swoją wybraną
USECOLOR="yes"

W /etc/locale.gen odznaczamy wszystkie wiersze zaczynające się na pl_PL poprzez usunięcie znaku hasha (#):

File: /etc/locale.gen
pl_PL ISO-8859-2
pl_PL.UTF-8 UTF-8

Na koniec, jako root, wydajemy polecenie:

# locale-gen

Możemy dodać także polecenie setxkbmap pl do pliku ~/.xinitrc (przed exec):

File: ~/.xinitrc
#!/bin/sh

if [ -d /etc/X11/xinit/xinitrc.d ]; then
  for f in /etc/X11/xinit/xinitrc.d/*; do
    [ -x "$f" ] && . "$f"
  done
  unset f
fi

setxkbmap -option terminate:ctrl_alt_bksp 
setxkbmap pl

exec gnome-session

Jeżeli ustawiłeś wszystko, jak trzeba (informacje powyżej i te w dalszych rozdziałach), a dalej nie masz spolszczonego systemu - spróbuj jeszcze dodać do pliku ~/.bashrc wiersze:

File: ~/.bashrc
export LANG="pl_PL.UTF-8"
export LC_CTYPE="pl_PL.UTF-8"
export LC_NUMERIC="pl_PL.UTF-8"
export LC_TIME="pl_PL.UTF-8"
export LC_COLLATE="pl_PL.UTF-8"
export LC_MONETARY="pl_PL.UTF-8"
export LC_PAPER="pl_PL.UTF-8"
export LC_NAME="pl_PL.UTF-8"
export LC_ADDRESS="pl_PL.UTF-8"
export LC_TELEPHONE="pl_PL.UTF-8"
export LC_MEASUREMENT="pl_PL.UTF-8"
export LC_IDENTIFICATION="pl_PL.UTF-8"
export LC_ALL="pl_PL.UTF-8"

[edytuj] Xorg

[edytuj] W wersji 1.7 i niższej

  • HAL

Po upewnieniu się, iż HAL jest zainstalowany i uruchomiony, wpisujemy w terminalu następujące polecenie:

# cp /usr/share/hal/fdi/policy/10osvendor/10-keymap.fdi /etc/hal/fdi/policy/

Otwieramy edytorem tekstowym plik /etc/hal/fdi/policy/10-keymap.fdi i zmieniamy wartość w wierszu input.xkb.layout na pl. Możemy zrobić tą samą operację z wierszem input.xkb.variant. Zostaje nam jeszcze:

# /etc/rc.d/hal restart
  • Xorg

Edytujemy /etc/X11/xorg.conf. W sekcji InputDevice, określającej klawiaturę, powinny być takie wpisy:

File: /etc/X11/xorg.conf
Section "InputDevice"
 ...
 Option "XkbRules"  "xorg"
 Option "XkbModel"  "pc105"
 Option "XkbLayout" "pl"
EndSection

W razie problemów możemy dodać także do /etc/X11/xorg.conf poniższą sekcję:

File: /etc/X11/xorg.conf
Section "ServerFlags"
 Option  "AutoAddDevices"  "False"
EndSection

[edytuj] W wersji 1.8 i wyższej

Od wersji 1.8 Xorg korzysta do rozpoznawania sprzętu z mechanizmu udev zamiast hal i nie korzysta już z ustawień w pliku /etc/X11/xorg.conf czy /etc/hal/fdi/policy/10-keymap.fdi.

Tak więc aby mieć polskie znaki, należy utworzyć/edytować /etc/X11/xorg.conf.d/10-keyboard.conf, a następnie upewnić się, że mamy podobny wpis:

File: /etc/X11/xorg.conf.d/10-keyboard.conf
Section "InputClass"
    Identifier            "Keyboard Defaults"
    MatchIsKeyboard       "yes"
    Option                "XkbModel" "pc105"
    Option                "XkbLayout" "pl"
    Option                "XkbOptions" "terminate:ctrl_alt_bksp"
 EndSection

[edytuj] Środowiska graficzne

[edytuj] GNOME

  • Język polski:
    • Samoczynnie (ewentualnie: patrz rozdział: Xorg).
  • Układ klawiatury:
    • Samoczynnie (ewentualnie Ustawienia systemu > Region i język)
  • Aby mieć w oknie GDM ustawienia polskie, należy wyedytować plik ~ /.drmc, aby wyglądał jak na poniższym przykładzie:
File: ~ /.drmc
[Desktop]
Language=pl_PL.UTF-8
Layout=pl
Notka: Poprawna konfiguracja GDM rozwiązuje problem angielskiego języka klawiatury, pojawiającego się na panelu.

[edytuj] KDE

Instalujemy polski pakiet językowy:

# pacman -S kde-l10n-pl

Następnie poprzez menu KDE:

  • Język polski:
    • Settings → System Settings → Regional & Language → Country/Region → Country Europe, Central → Poland → Add Language → Polish
    • → Remove → US English
  • Układ klawiatury:
    • Settings → System Settings → Regional & Language → Keyboard Layout
    • → Zaznacz "Enable keyboard layout" → Available layouts → Poland → Kliknij na zieloną strzalkę w prawo przy Available layouts → Apply

[edytuj] Sprawdzenie poprawnego spolszczenia

Aby sprawdzić, czy system jest spolszczony, użyj polecenia locale. Powinieneś otrzymać wynik, jak poniżej:

$ locale
LANG=pl_PL.UTF-8
LC_CTYPE="pl_PL.UTF-8"
LC_NUMERIC="pl_PL.UTF-8"
LC_TIME="pl_PL.UTF-8"
LC_COLLATE="pl_PL.UTF-8"
LC_MONETARY="pl_PL.UTF-8"
LC_MESSAGES="pl_PL.UTF-8"
LC_PAPER="pl_PL.UTF-8"
LC_NAME="pl_PL.UTF-8"
LC_ADDRESS="pl_PL.UTF-8"
LC_TELEPHONE="pl_PL.UTF-8"
LC_MEASUREMENT="pl_PL.UTF-8"
LC_IDENTIFICATION="pl_PL.UTF-8"
LC_ALL=pl_PL.UTF-8
