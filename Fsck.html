[edytuj] Wprowadzenie

Fsck jest to uniksowy program do sprawdzania integralności systemu plików. Zazwyczaj uruchamiany jest podczas bootowania systemu, aby sprawdzić, czy system plików jest w dobrym stanie.

W Arch Linuksie Fsck sprawdza w sposób zaplanowany system plików, także użytkownik nie jest zobowiązany, aby o tym pamietać.

[edytuj] Manualne uruchomienie narzędzia

W Arch Linuksie można uruchomić pełne sprawdzanie fsck w bardzo prosty sposób za pomocą jednego polecenia. Po pierwsze, wyjdź z środowiska graficznego lub menedżera okien, zaloguj się na root (#) i wpisz następujące polecenie:

# shutdown -Fr now

Po wydaniu polecenia system powinien zrestarować się, po czym podczas uruchamiania się Archa fsck powinien sprawdzić Twoje partycje.

[edytuj] Częstotliwość kontroli

Narzędzie fsck domyślnie sprawdza partycje co 30 uruchomień (liczonych indywidualnie dla każdej partycji). Aby zmienić częstotliwość sprawdzania, wykonaj:

# tune2fs -c 20 /dev/sda1

Liczba 20 oznacza, po ilu bootowaniach systemu narzędzie będzie sprawdzało systemy plików. Po liczbie wpisana jest partycja, której dotyczy to polecenie.

Ponadto:

  • 1 oznacza sprawdzanie podczas każdego yrychomienia systemu
  • 0 wyłącza fsck

Jeśli chcesz sprawdzić liczbę i częstotliwość bieżącego odliczania, użyj polecenia:

# dumpe2fs -h /dev/sda1 | grep -i 'mount count'

[edytuj] Przydatne informacje

Jeżeli jesteś ciekaw listy wszystkich dostępnych opcji, wpisz polecenie:

# fsck-h

Aby narzędzie automatycznie naprawiało uszkodzone systemy plików:

# fsck -a

Przeciwnością powyższego jest następujące polecenie:

# fsck -n

[edytuj] Narzędzie Fsck, a plik konfiguracyjny /etc/fstab

W pliku /etc/fstab w szóstej kolumnie każdego wiersza z montowaną partycją masz podaną wartość, która informuje system, czy dana partycja ma być sprawdzana czy też nie: 

/dev/sda1 / ext4 defaults 0 1
/dev/sda2 /other ext4 defaults 0 2
/dev/sda3 /win ntfs defaults 0 0
  • 0 = wyłącza sprawdzanie
  • 1 = pierwszy system plików (partycja) do kontroli; wpis montujący / powinien być ustawiony na 1
  • 2 = wszystkie inne systemy plików, które należy sprawdzić.
